Black-Karasinski model for short interest rate

$$d(\ln r_t) = (a(t) - \theta(t) \ln r_t)dt + \sigma(t) dW(t)$$

is solved below using Euler scheme for stochastic differential equations

In [4]:
def Euler(a, theta, sigma):

    X = np.zeros(n+1, dtype=np.float64)

    h_sqrt = np.sqrt(h)
    X[0] = x_0

    for i in range(1, n+1):
        dW = np.random.normal(0, h_sqrt)
        X[i] = X[i-1] + (theta-a*X[i-1])*h + sigma*dW
    return  np.exp(X)

Price market $$P^M(0,T) = \mathbb{E}\Big(e^{-\int\limits_0^Tr_t dt}\Big) \approx \sum_{i=1}^m \exp\left({-h\sum_{j=1}^n r_{t_j}}\right)$$

k = 100
maturities = range(10,100,10)

def P_M(a, theta, sigma):

    P = []
    for j in maturities:
        #t = [i*h for i in range(j+1)]
        temp = np.zeros(k)
        for i in range(k):
            temp[i] = np.exp(-np.sum(Euler(a, theta, sigma)[:j+1])*h)
        P.append(np.mean(temp))
    return P

def J(X):
    a_ = X[0]
    theta_ = X[1]
    sigma_ = X[2]
    return np.sum((generated_prices - P_M(a_, theta_, sigma_))**2)
